# ICT Game Jam Winter 2026

A shooter game made for ICT Game Jam Winter 2026. by Charlie @cammymoop and John @jvishnefske


### Controls
| Input | Action |
|-------|--------|
| WASD | Move player |
| Mouse | Aim cursor |
| Left Click | Fire projectile |

### Run Tests
```bash
# Requires Godot in PATH or set GODOT env var
make test
```
